# eext-schematic-pdf-interaction

原理图PDF交互扩展 — 在 PCB 编辑器中预览原理图 PDF，点击元件位号文本自动选中高亮并定位 PCB 封装。非常适合只有原理图PDF又需要和PCB对位的场景。

## 功能

- 仅在 PCB 编辑器中显示菜单入口（顶部菜单 → 原理图PDF交互 → 打开原理图PDF...）
- 点击菜单打开 IFrame 弹窗（支持最小化），选择本地 PDF 文件（点击或拖拽）
- 点击"预览 PDF"按钮，在独立浏览器窗口中渲染 PDF
- 点击 PDF 中的元件位号文本（如 R1、C2、U3），PCB 自动选中高亮对应封装
- 三个用户选项：
  - ☑ 自动收起弹窗（默认开启）— 点击预览后自动最小化弹窗
  - ☑ 自动平移（默认开启）— 点击位号后平移画布到元件中央
  - ☐ 自动缩放定位（默认关闭）— 点击位号后缩放并定位到元件
- PDF 预览支持：翻页、缩放（按钮 / Ctrl+滚轮）、适合宽度、书签/大纲导航
- 关闭预览窗口后自动恢复 IFrame 弹窗
- 位号未找到时显示明确提示
